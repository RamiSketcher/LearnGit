## git
status <br>
- info about repo* <br>
add (file name) <br>
- prepare file to comiit <br>
commit -m "good commit message" <br>
- save added files <br>
- -m: add message about the commit <br>
push <br>
- publish or share <br>
log <br>
- general log for all commits <br>

*repo: every file and every folder in the project
